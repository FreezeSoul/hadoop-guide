## Hive

### （一）什么是 Hive

* 构建在 HDFS 之上的一个**数据仓库**

* 同时也是一个支持SQL语言的**数据分析引擎**

* 在 Hive 上执行的复杂的 SQL 将会被转换成一个 MapReduce 程序提交到 Yarn 上运行

* Hive 的数据存储在 HDFS 上，表的元信息（表名，字段名，字段类型，分区信息）存储在 MySql 中
